### 평가 방법

#### 이동 평균

- 원자재 그대로 파는 경우에 쓰임. 반제품같은 경우에는 원재료에 공정이 들어가서 제품로 생산하여 원가의 변동이 발생하므로 쓰이지 않는다.

- 각각 날짜의 단가 * 수량 = 재고 금액

- | 1/1  | 5        | 100  | 500  |
  | ---- | -------- | ---- | ---- |
  | 1/2  | 5        | 200  | 1000 |
  | 1/3  | 납품시작 |      |      |

- 1/3 납품 5개.의 평균 단가

- 매출원가계정 : (500 +     1000) / 재고 개수 10 = 150

- 이동 균 = 납품 개수 * 이동 평균 단가 150 = 750

 

#### 표준 원가

- 한 달 단위로 쓰인다. 반제품, 제품에 쓰인다.
- 품목에 쓰임.
- 품목 원가 * 수량
